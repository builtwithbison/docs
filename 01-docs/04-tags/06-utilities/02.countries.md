{{ noparse }}

Loops through all available countries. [Read about locale settings](/docs/settings/locales)

### Parameters
`sort_by`
: Sort by either country `name`, country `code` or leave this parameter out to sort by the order in your locales file.

### Variables
`country`
: Full country name

`country_code`
: 2 or 3 character country code


### Example
~~~
<ul>
{{ bison:countries }}
  <li>{{ country_code }}: {{ country }}</li>
{{ /bison:countries }}
</ul>
~~~
